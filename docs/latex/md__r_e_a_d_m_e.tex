A C++23 3D engine built with Mini\+FB as single dependency. Targets Unix environments.\hypertarget{md__r_e_a_d_m_e_autotoc_md69}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md69}

\begin{DoxyItemize}
\item OBJ file parsing via C++23 named modules ({\ttfamily import \doxylink{classobj}{obj}}) built from scratch.
\item Real-\/time windowed rendering with \href{https://github.com/nicowillis/minifb}{\texttt{Mini\+FB}}
\item Simple 3D rendering pipeline built from scratch.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md70}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md__r_e_a_d_m_e_autotoc_md70}

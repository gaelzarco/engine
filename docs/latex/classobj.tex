\doxysubsection{Detailed Description}
Represents a parsed Wavefront OBJ file. 

Stores all geometric data defined by the OBJ specification, including vertices, texture coordinates, vertex normals, parameter space vertices, faces, and lines. Provides a high-\/performance parser using {\ttfamily std::\+from\+\_\+chars} for zero-\/allocation numeric conversion.

\begin{DoxyNote}{Note}
Exported from C++20 module {\ttfamily obj\+\_\+v0\+\_\+3}.
\end{DoxyNote}

Parses a Wavefront OBJ file and populates the model\textquotesingle{}s geometry data. 

Reads the entire file into a buffer and performs a single-\/pass parse using {\ttfamily std::\+from\+\_\+chars} for locale-\/independent, allocation-\/free numeric conversion. Recognised keywords:\+ {\ttfamily v}, {\ttfamily vt}, {\ttfamily vn}, {\ttfamily vp}, {\ttfamily f}, {\ttfamily l}. Comment lines ({\ttfamily \#}) and blank lines are skipped. Unrecognised keywords are silently ignored and the remainder of the line is consumed.

Face vertex tokens support all three OBJ index formats:\+
\begin{DoxyItemize}
\item {\ttfamily v} 
\item {\ttfamily v/\+vt} 
\item {\ttfamily v/\+vt/\+vn} 
\item {\ttfamily v/\+/\+vn} 
\end{DoxyItemize}

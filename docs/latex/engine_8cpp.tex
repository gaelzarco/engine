\doxysubsection{Detailed Description}
Entry point for the CXX Rasterizer — a minimal Wavefront OBJ viewer with a software-\/rendered Mini\+FB window. 

Loads a Wavefront OBJ file supplied on the command line, logs parse time and heap usage, opens an 800×600 resizable Mini\+FB window, and renders a small set of test lines using {\ttfamily render\+\_\+v0\textquotesingle{}s} Bresenham rasterizer before entering the main display loop.

\begin{DoxyParagraph}{Dependencies}

\begin{DoxyItemize}
\item C++20 module {\ttfamily obj\+\_\+v0\+\_\+3} — OBJ file parsing and in-\/memory representation.
\item C++20 module {\ttfamily render\+\_\+v0} — 2D primitive rasterization into a pixel buffer.
\item Mini\+FB — cross-\/platform framebuffer window library.
\end{DoxyItemize}
\end{DoxyParagraph}

Application entry point. 

Performs the following steps in order:\+
\begin{DoxyEnumerate}
\item Validates that a file path argument was supplied; exits with code 1 otherwise.
\item Parses the OBJ file at {\ttfamily argv}\+[1]\+, logging wall-\/clock parse time and reserved heap memory via {\ttfamily \doxylink{classobj_aa2ec83bc4e4ef053ad37a4da935c747e}{obj::\+memory\+\_\+out()}}.
\item Opens a resizable Mini\+FB window titled "{}\+CXX Rasterizer"{} at {\ttfamily WIDTH} × {\ttfamily HEIGHT} pixels; exits with code 1 on failure.
\item Allocates a flat {\ttfamily WIDTH} × {\ttfamily HEIGHT} pixel buffer initialised to zero.
\item Constructs three test points and four randomly coloured {\ttfamily \doxylink{structbuff__color}{buff\+\_\+color}} instances, then draws four lines between those points into the buffer using {\ttfamily render\+\_\+v0::\+line()}.
\item Enters the Mini\+FB event loop, pushing the buffer to the window every sync interval until the window is closed or an error state is returned.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
The four {\ttfamily \doxylink{structbuff__color}{buff\+\_\+color}} instances are default-\/constructed (all channels 0), which triggers random colour generation in {\ttfamily \doxylink{structbuff__color}{buff\+\_\+color}\textquotesingle{}s} constructor. Each line is therefore drawn in a distinct, unpredictable colour.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}

\end{DoxyWarning}

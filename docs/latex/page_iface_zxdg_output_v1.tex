An xdg\+\_\+output describes part of the compositor geometry.

This typically corresponds to a monitor that displays part of the compositor space.

For objects version 3 onwards, after all xdg\+\_\+output properties have been sent (when the object is created and when properties are updated), a wl\+\_\+output.\+done event is sent.
